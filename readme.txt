== Description ==
Watermark PDF for WordPress and WooCommerce is an easy-to-use plugin that lets you add text and image watermarks to PDF files to protect them from unauthorized use. With this plugin, you can customize the font size, font color, font family, opacity, and repeat for text watermarks, and width, height, opacity, and rotation for image watermarks.

== Features ==
* Add text and image watermarks to PDF files for enhanced security
* Customize font size, color, family, opacity, repeat for text watermarks
* Customize width, height, opacity, and rotation for image watermarks

== Requirements ==
The plugin requires [Ghostscript](https://www.ghostscript.com/) package installed on your server to handle PDF files that have advanced compression technique. You can check if the package is installed at your server or not from settings page.


== Limitations ==
The plugin is limited to watermarking PDF files which don't have input forms, annotations or bookmarks.
